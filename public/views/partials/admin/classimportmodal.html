Import Classes from CSV File

Upload a CSV file with class data. The CSV file should have column headers for "Name"*, "Class ID", "Show Dates", "Show Grades", "Show Grades As Score", "Show Grades As Letter", "Show Grades As Percentage", "User IDs"

Class ID field is optional when adding a new class but is required when editing an existing class. If a class with the given ID does not exist, the class will be added to the system. If a class with the provided ID already exists, that class information will be updated with values from any non-blank columns. Fields marked with * are required. User IDs column is a semicolon separated list of either internal or external user IDs of existing users to enroll in the system.
